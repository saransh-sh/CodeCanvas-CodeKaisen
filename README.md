# CodeCanvas – Personal Productivity and Habit Tracker

**Team:** CodeKaisen

## About

CodeCanvas is a personal productivity and habit tracker application designed to help users build consistent habits, manage daily tasks, and track their progress over time.

## Features

- Track daily habits and streaks
- Manage tasks and to-do lists
- Visualize productivity progress
- Simple and intuitive interface

## Getting Started

Clone the repository and follow the setup instructions to get started:

```bash
git clone https://github.com/saransh-sh/CodeCanvas-CodeKaisen.git
cd CodeCanvas-CodeKaisen
```
